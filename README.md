Hello everyone this repository is my attempt to make bash script installer where this script can be used to install monitoring services for your server

This installer include
1. Prometheus (node exporter included for exporting data of your node ex: CPU usage & RAM usage )
2. Grafana ( to visualize the data that has been collected by prometheus for easy reading )

NOTE:
THE PROMETHEUS CONFIGURATION FILE (prometheus.yml) IT CAME UNMODIFIED, PLEASE MODIFY TO YOUR NEEDS
